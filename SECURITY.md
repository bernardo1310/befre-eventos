# Política de Segurança

## Versões Suportadas

Este projeto está em desenvolvimento ativo.

## Relatando Vulnerabilidades

Se você encontrar uma vulnerabilidade de segurança, por favor, não abra uma issue pública. Em vez disso, entre em contato através de:

- Email: bernardo@befre-eventos.com (ou seu email de contato)
- GitHub Security Advisory

Por favor, inclua:
- Uma descrição da vulnerabilidade
- Passos para reproduzir
- Possível impacto

## Medidas de Segurança Implementadas

- ✅ HTTPS (via GitHub Pages)
- ✅ Content Security Policy (CSP)
- ✅ X-Frame-Options
- ✅ X-Content-Type-Options
- ✅ Referrer-Policy
- ✅ Permissions-Policy
- ✅ Validação de entrada
- ✅ Proteção contra XSS
- ✅ Proteção contra CSRF (quando aplicável)
